我的帥氣技術筆記

安全提醒： 網路工具（如 Nmap、Tcpdump）應用於經授權的網路管理和安全測試。未經許可的掃描或攔截行為是違法且不道德的。

1. 確認自己的 IP 位置

ip addr

2. 找出 DMZ 內所有開啟的 Services (Port)

nmap {你的IP位置}

3. 嘗試連接內部電腦

ssh {使用者名稱}@{主機IP}

4. 攔截帶有帳號密碼的封包

tcpdump

5. 解碼封包

base64 -d {    }

6. 登入內部主機

ssh {    }@{    }

7. 利用linux指令操作內部主機

ls

8. 讀取重要資料

cat {    }

9. 斷開內部電腦

logout

10. 送出資料

mailto ~[1130730@fheh.isip.org](mailto:1130730@fheh.isip.org)~ {    }
